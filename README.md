# Project Vega

lessgoo.

Project Vega is an agent workspace and monitoring system for software engineering. The product goal is to give developers a simple, opinionated way to manage parallel coding-agent work without forcing them to manually juggle transcript logs, diffs, terminals, review context, and task state across a pile of tabs.

This repository currently contains the initial architecture scaffold and the core planning docs for the first build.
